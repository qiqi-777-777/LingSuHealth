## 新增功能详解

- 入睡时间（必填）

**步骤2：健康状态**
- 今日症状
- 情绪状态（必填）
- 运动时长

**步骤3：饮食记录**
- 饮食简记

#### 风险提示

## 使用说明

### 填写记录
1. 点击"开始记录"
2. 按步骤填写信息
3. 必填项会有红色星号标识
4. 点击"下一步"自动验证
5. 完成后提交

### 查看历史
1. 右侧面板显示近7天记录
2. 点击筛选按钮展开筛选器
3. 选择筛选条件
4. 点击"应用筛选"

### 导出记录
1. 筛选想要导出的记录
2. 点击"导出PDF"按钮
3. 选择导出范围
4. 等待生成完成
5. 自动下载PDF文件

## 技术栈
- Vue 3 Composition API
- TypeScript
- html2canvas（截图）
- jsPDF（PDF生成）
- Chart.js（图表）

## 注意事项
1. 必填字段未填写时无法进入下一步
2. PDF 导出需要 1-3 秒时间
3. 筛选条件会保存在本地
4. 移动端自动适配布局
