# 测评功能使用指南

## ✅ 已完成的优化

### 1. 进度保存与恢复
- **自动保存**：答题时自动保存进度，无需手动操作
- **智能恢复**：重新打开页面时，如果有未完成的测评（24小时内），会弹出提示询问是否继续
- **一键清除**：可以选择放弃之前的进度，重新开始

### 2. 题目导航
- **快速跳转**：长测评（25题）会显示题目导航栏
- **状态标记**：
  - 绿色边框 = 已回答
  - 绿色填充 = 当前题目
  - 灰色 = 未回答
- **点击跳转**：直接点击题号即可跳转到对应题目

### 3. 结果展示优化
结果页面重新组织为4个独立区块：

#### 区块1：评估结论
- 体质类型（大字显示）
- 体质简述说明

#### 区块2：健康风险提示
- 列表展示所有风险项
- 红色警告图标突出显示

#### 区块3：调理建议
- 编号列表展示建议
- 包含饮食、运动、作息等方面
- 根据体质类型自动生成

#### 区块4：健康画像数据
- 可视化图表（健康五维分析、体质倾向分布）
- 睡眠、情绪、饮食数据卡片

### 4. 导出功能
- **单独导出**：每个区块右上角有"导出"按钮
- **完整导出**：顶部"导出报告"按钮可导出全部内容
- **格式**：纯文本（.txt），包含时间戳

### 5. 分享功能
- 点击"分享"按钮
- 复制链接到剪贴板
- 可分享给他人查看

## 🎯 使用流程

### 开始测评
1. 进入测评页面
2. 如果有未完成的测评，选择"继续测评"或"放弃"
3. 选择"快速测评"（10题）或"深度测评"（25题）

### 答题过程
1. 单选题：点击选项即可
2. 多选题：可选择多个选项
3. 使用"上一题"/"下一步"按钮导航
4. 长测评可使用顶部的题目导航快速跳转
5. 进度自动保存，随时可以离开

### 查看结果
1. 完成所有题目后点击"提交测评"
2. 查看四个区块的详细结果
3. 可以单独导出某个区块，或导出完整报告
4. 点击"分享"复制链接分享给他人

## 💡 提示

- 进度保存有效期为24小时
- 提交测评后会自动清除保存的进度
- 建议使用Chrome、Edge、Safari等现代浏览器
- 导出的文件编码为UTF-8，可用记事本或Word打开

## 🔧 技术说明

- 使用localStorage存储进度数据
- 数据仅保存在本地浏览器，不会上传到服务器
- 清除浏览器数据会同时清除保存的进度
